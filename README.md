# Creative Experiments

Design experiments and small built works — a public-safe sandbox.

## Start here

Open [`creativevault_thought_gallery_rfm_v0_1/index.html`](creativevault_thought_gallery_rfm_v0_1/index.html) — a small gallery site that links most of the other pieces.

## Highlights

- **Reference-first mimic (RFM)** — a gated workflow for reference-led builds. Route: `reference_first_mimic_workflow_v0_2.md`. Web test builds under `reference_first_mimic_test_v0_*/` and `rfm_codex_test_v0_1/`.
- **Posters and key visuals** — Hospital Year-End Party 2026, Creative Signal Night, Tomorrow Less Boring, Memory Garden.
- **Template** — `_rfm_template/` is the per-build scaffold (`00_GATES.md` is the gate checklist).

## Notes

Everything here is intentionally non-confidential. No medical, patient, or unpublished material; no secrets, API keys, or tokens.
