# Reference Observation — Reference-first mimic test v0.3

Created: 2026-06-12
Method: references decomposed at a structural level from their well-known design
language. No source code, copy, imagery, fonts, colours, or layout copied. No
reference screenshots saved into this project.

Direction this round: **bright, maximalist, designed pop** — explicitly NOT v0.1
(riso / registration offset / newsprint / giant-type) and NOT v0.2 (photo archive /
finding aid / quiet paper). Pop, but not a cheap AI banner: no pastel-only, no
emoji-scatter, no round/pill reliance, no even card grid, no cute-but-weak.

## 1. Candidates (real, strong, bright/playful — not SaaS, not card-UI)

### A. Morag Myerscough — moragmyerscough.com  ★ lead
- **Why good:** Joyful maximalist signage / installation design. Enormous bold
  lettering on saturated colour blocks, dense pattern fills, asymmetric hand-built
  compositions. Bright but rigorous — never childish, never pastel-cute.
- **Borrow:** big type locked into colour blocks; saturated colour-blocking as the
  page structure; pattern fills (stripes / zigzag / checker); celebratory density.
- **Don't borrow:** her exact palettes, slogans, installation photos, letterforms.

### B. Camille Walala — camillewalala.com  (pattern / geometry reference)
- **Why good:** Memphis-revival geometric pop — bold stripes, triangles, zigzags,
  chevrons, primary-leaning colour. Pattern as identity, almost no reliance on circles.
- **Borrow:** angular pattern vocabulary (stripes, triangles, zigzag) so "pop" does
  not collapse into dots-and-pastels; confident clashing colour.
- **Don't borrow:** her specific murals, colour combos, mascot shapes.

### C. Hey Studio — heystudio.es  (icon / flat-shape reference)
- **Why good:** Clean geometric illustration system, bold flat colour, precise
  shape-building, strong grids broken playfully. Pop with discipline.
- **Borrow:** building illustrations from a few precise flat geometric shapes;
  restrained shadow; crisp die-cut sticker feel.
- **Don't borrow:** their icon set, brand work, colour, type.

## 2. Chosen lead + blend

Lead: **Morag Myerscough** (colour-block + big type + pattern = the page's bones).
Pattern vocabulary: **Camille Walala** (stripes / triangles / zigzag, not dots).
Sticker / flat-shape craft: **Hey Studio**.

## 3. Decomposition of the lead (and where pop goes cheap)

- **First view:** NOT a centred hero. The screen is split into clashing colour
  blocks; huge type sits inside a block, off-centre, with a built (not photographed)
  object overlapping the seam and angled stickers breaking the edges.
- **Colour:** large saturated colour FIELDS do the structural work (block per zone),
  with sharp clashing accents — not pastel washes, not one flat white background.
- **Shapes / stickers:** die-cut labels, spiky burst seals, banners and cut wedges,
  placed unevenly and rotated — overlapping block seams, not sitting in a tidy grid.
- **Whitespace:** "weird" — tight dense clusters of stickers next to deliberately
  empty colour fields. Density varies hard between zones.
- **Typography:** a heavy quirky grotesque for structure + a fat display face for
  loud labels + mono for the "lab data" layer (numbers, %, batch codes).
- **Grid-breaking:** elements bleed across colour-block seams and off the page edges;
  rotations everywhere; the pattern fills ignore the text grid.
- **Information density:** multiple registers on one screen — giant flavour names,
  small lab readouts, a tiny ticker, a bar chart — so it reads designed, not cute.
- **vs generic AI LP:** colour-blocking + pattern + uneven stickers instead of white
  bg + even cards; loud type instead of eyebrow→H1→subtitle→pill; the same loud
  language runs top-to-bottom.
- **Where it goes CHEAP if mimicked badly:** pastel + rounded-everything + emoji +
  evenly-spaced cute cards + one drop-shadow on white. Avoided by: saturated (not
  pastel) colour, angular pattern (not dots), a pseudo-scientific data layer for
  rigour, hard asymmetry, and a print-grain texture so it isn't flat-digital.
